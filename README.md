# Contador para apresentações

Um dos desafios da apresentação do meu TCC, é condensar uma grande quantidade de informações (mais especificamente 59 páginas) em 30 minutos de apresentação, então para me organizar melhor, criei este contador para a minha apresentação.

Atualmente os dados são setados de forma manual no código através de uma estrutura dessa maneira:

```jsx
  const contadores = [
    { titulo: "Introdução", tempo: 3 },
    { titulo: "Objetivos", tempo: 2 },
    { titulo: "Fundamentação", tempo: 3 },
    { titulo: "Hardware", tempo: 5 },
    { titulo: "Software", tempo: 3 },
    { titulo: "Encapsulamento", tempo: 2 },
    { titulo: "Resultados", tempo: 5 },
    { titulo: "Trabalhos futuros", tempo: 1.5 },
    { titulo: "Conclusão", tempo: 1.5 },
  ];
```

É um projeto de uso pessoal, por isso os dados são setados manualmente, mas ainda são fáceis de serem editados, podendo remover ou adicionar novos sem nenhum problema.
